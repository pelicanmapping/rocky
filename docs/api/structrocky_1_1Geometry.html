struct Geometry

Geometric shape data.

This class holds one or more lists of points. How these points are interpreted depends on the geometry Type.

Public Types

enum class Type

Types of geometry supported in this class.

Values:

enumerator Points

A single collection of points (in “points”)

enumerator LineString

A single collection of line segments (in “points”)

enumerator Polygon

One polygon.

“points” contains the outer ring with a CCW winding. “parts” contains an optional collection of holes, each with a CW winding.

enumerator MultiPoints

Each part contains one or more points; “points” is ignored.

enumerator MultiLineString

Each part contains a separate linestring; “points” is ignored.

enumerator MultiPolygon

Each part contains a separate polygon; “points” is ignored.
