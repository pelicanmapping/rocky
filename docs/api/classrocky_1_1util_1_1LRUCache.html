Class Documentation

template<class K, class V>
class LRUCache : public rocky::Cache<K, V>

LRUCache is a thread-safe implementation of a Least Recently Used (LRU) cache.

It stores key-value pairs and evicts the least recently used item when the cache reaches its capacity. The cache is protected by a mutex to ensure thread safety. Adapted from https://www.geeksforgeeks.org/lru-cache-implementation Optimized by Copilot

If the key already exists, updates its value and moves it to the most recently used position. If the cache is full, evicts the least recently used item.
