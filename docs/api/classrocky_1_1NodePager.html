Class Documentation

class NodePager : public vsg::Inherit<vsg::Group, NodePager>

Node that manages a dynamically paged scene graph.

The graph’s structure is based on a Profile and each represents a TileKey in that Profile.

To use a NodePager you must set the context, profile, and createPayload function.
