Detailed Description

template<typename F>
class ROCKY_NAMESPACE::Callback< F >

Easy way to add a thread-safe callback to a class.

Developer defines a callback, usually as a class member: Callback<void(int)> onClick;

User adds a callback: instance->onClick([](int a) { ... });

Class fires a callback: onClick(a);
