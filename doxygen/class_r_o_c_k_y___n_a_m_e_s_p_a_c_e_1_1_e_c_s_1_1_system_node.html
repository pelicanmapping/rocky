Detailed Description

template<class T, class R>
class ROCKY_NAMESPACE::ECS::SystemNode< T, R >

VSG node representing an ECS system for the given component type (T) and corresponding renderable type (R). This lives under a SystemsManagerGroup node.

Parameters
TThe component type
RThe renderable type (a struct that contains drawable data if necessary)
