Crops the image to a new geospatial extent.

Parameters
extentNew extent to which to crop the image.
exactIf "exact" is true, the output image will have exactly the extents requested; this process may require resampling and will therefore be more expensive. If "exact" is false, we do a simple crop of the image that is rounded to the nearest pixel. The resulting extent will be close but usually not exactly what was requested - however, this method is faster.
width,heightNew pixel size for the output image. By default, the method will automatically calculate a new pixel size.
