Detailed Description

Transform node that accepts geospatial coordinates and creates a local ENU (X=east, Y=north, Z=up) coordinate frame for its children that is tangent to the earth at the transform's geo position.

bool ROCKY_NAMESPACE::GeoTransform::localTangentPlane = true

Whether the transformation should establish a local tangent plane (ENU) at the position. Disabling this can increase performance for objects (like billboards) that don't need tangent plane.
