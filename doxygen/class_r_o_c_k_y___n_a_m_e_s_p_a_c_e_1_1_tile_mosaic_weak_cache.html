Detailed Description

template<class Value>
class ROCKY_NAMESPACE::TileMosaicWeakCache< Value >

A "cache" of weak pointers to values, keyed by tile key. This will keep references to data that are still in use elsewhere in the system in order to prevent re-fetching or re-mosacing the same data over and over.
