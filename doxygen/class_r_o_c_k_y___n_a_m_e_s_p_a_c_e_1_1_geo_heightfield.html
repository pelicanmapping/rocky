Samples the elevation value at a specified point.

Parameters
x,yCoordinates at which to query the elevation value.
srsSpatial reference of the query coordinates. (If you pass in NULL, the method will assume that the SRS is equivalent to that of the GeoHeightField. Be sure this is case of you will get incorrect results.)
interpInterpolation method for the elevation query.
srsWithOutputVerticalDatumTransform the output elevation value according to the vertical datum associated with this SRS. If the SRS is NULL, assume a geodetic vertical datum relative to this object's reference ellipsoid.
Returns
The elevation value, or NO_DATA_VALUE if the query failed
