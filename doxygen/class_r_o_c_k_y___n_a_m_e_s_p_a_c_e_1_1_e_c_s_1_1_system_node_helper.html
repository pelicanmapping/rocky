Detailed Description

template<class COMPONENT>
class ROCKY_NAMESPACE::ECS::SystemNodeHelper< COMPONENT >

Helper class for making systems that operate on Component types with VSG nodes in them. VSG nodes respond to many types of traversals like record, compute bounds, intersect... and this system-helper object will allow these traversals to hit those components even though they aren't in the scene graph. It will also facilitate grouping each component under the Graphics Pipeline appropiate for its rendering properties.
