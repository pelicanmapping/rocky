Detailed Description

Represents a single terrain tile in the scene graph, along with its potentially pageable children.

TileNode represents a single tile. TileNode has 5 children: one SurfaceNode that renders the actual tile content under a MatrixTransform; and four TileNodes representing the LOD+1 quadtree tiles under this tile.
